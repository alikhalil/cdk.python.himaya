# Himaya: Be Vigilant

This is an AWS CDK repository for the infrastructure to support himaya.tech

The infrastructure consists of:

* API Gateway
* Lambda Functions
    * Register order
    * Payment hook
* S3 Buckets
    * Static website hosting
    * Website access logs
    * Naked domain to www redirection
